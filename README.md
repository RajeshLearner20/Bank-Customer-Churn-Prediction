# 🏦 Bank Customer Churn Prediction

## 📌 Project Overview

This project predicts whether a bank customer will leave (churn) or stay using Machine Learning techniques.

## 🚀 Features

* User-friendly web interface using Django
* Customer data input form
* ML model prediction (Churn / Not Churn)
* Real-time prediction results

## 🛠️ Technologies Used

* Python
* Django Framework
* Machine Learning (Scikit-learn)
* HTML, CSS

## 📂 Project Structure

* `churnproject/` → Main Django project
* `churnapp/` → Core ML logic
* `userapp/` → User handling
* `templates/` → Frontend HTML files
* `static/` → CSS files
* `ml_model/` → Trained model

## ⚙️ Installation Steps

```bash
git clone https://github.com/RajeshLearner20/Bank-Customer-Churn-Prediction.git
cd Bank-Customer-Churn-Prediction
pip install -r requirements.txt
python manage.py runserver
```

## 📊 Output

* Predicts customer churn based on input data
* Displays result instantly on the webpage
